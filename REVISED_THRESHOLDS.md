## NEW THRESHOLDS:

### Team Insights:
**Win Percentage:**
- 77%+ → "Dominant force" (Top 1-2 teams only)
- 70-77% → "Strong contender" (Ranks 2-5)
- 60-70% → "Competitive team" (Ranks 6-10)
- <60% → "Building momentum"

**Points Per Match:**
- 22+ → "High-scoring team" (Top 3 only)
- 19-22 → "Solid performer" (Ranks 4-8)
- 16-19 → "Consistent team" (Middle pack)

**Rank:**
- ≤2 → "Elite championship team"
- 3-5 → "Top-tier contender"
- 6-12 → "Mid-table competitor"

**Total Points:**
- 190+ → "Point machine" (Top 3 only)
- 170-190 → "Strong accumulator" (Ranks 4-7)

### Player Insights:

**Batting - Total Runs:**
- 180+ → "Leading run-scorer" (Top 5 only)
- 140-180 → "High-impact batsman" (Ranks 6-15)
- 110-140 → "Solid contributor"

**Batting - Strike Rate:**
- 120+ → "Aggressive explosive striker" (Top 3 only)
- 105-120 → "Balanced attacking batsman" (Ranks 4-10)
- 90-105 → "Steady accumulator"

**Batting - Average:**
- 35+ → "Elite consistency" (Top 5%)
- 25-35 → "Key contributor" (Top 20%)
- 18-25 → "Solid performer"

**Bowling - Wickets:**
- 13+ → "Leading wicket-taker" (Top 5 only)
- 10-13 → "Strike bowler" (Ranks 6-15)
- 8-10 → "Consistent contributor"

**Bowling - Economy:**
- <4.0 → "Exceptional economy" (Top 5 only)
- 4.0-4.8 → "Economical bowler" (Top 15)
- 4.8-5.5 → "Reliable bowler" (Average)
- >5.5 → "Attacking approach"

### Match Strategy:

**Bowling threat levels:**
- 13+ wickets → "Elite threat - play very defensively"
- 11-13 wickets → "Dangerous bowler - be cautious"
- 8-11 wickets → "Quality bowler - respect early"

**Economy ratings:**
- <4.2 → "Extremely tight - rotate strike carefully"
- 4.2-5.0 → "Economical - be patient"
- >5.5 → "Target aggressively"

**Batting threat levels:**
- 180+ runs → "Division star - get them early"
- 150-180 runs → "Dangerous batsman - set attacking fields"
- 140-150 runs → "Key player - bowl tight lines"

**Team strength:**
- 77%+ win rate → "Elite team - peak performance required"
- <35% win rate → "Capitalize on weaknesses"
- Rank ≤2 → "Championship team - focus on fundamentals"
